# BBC ambience files

Place the three downloaded MP3 files in this folder with these exact names:

- `bbc_song-thrus_nhu0508018.mp3` — displayed in the app as **Song thrush birdsong**
- `bbc_fire---clo_nhu0500210.mp3` — displayed in the app as **Close fire / fireplace**
- `bbc_rain---hea_nhu0503213.mp3` — displayed in the app as **Heavy rain**

The app reads these paths from `assets/data/audio-library.json`.

The files are not included in this generated package. Add your own downloaded files, then commit them if your intended use/licence allows it.
